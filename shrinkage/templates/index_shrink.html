Description

Upload data as a CSV file with concentrations in LV%.

This Model uses machine learning (a feed forward neural network) to predict the shrinkage of the sample. It uses over 7000 training points based on real data to estimate shrinkage at standard temp and pressure (60 f and 0 psig).

Important!

The columns names in the csv file need to contain: LV % N2, LV % CO2, LV % C1, LV % C2, LV % C3, LV % IC4, LV % NC4, LV % IC5, LV % NC5, LV % C6, LV % C7, C7+ Specific Gravity, C7+ Molecular Weight, Temp, Pressure, and Shrinkage Factor.

** if you are using this template to find shrinkage instead of compare shrinakges set the shrinkage values in the "Shrinkage Factor" column to 1

This code will not work if the column names don't match!
